# MTN Nigeria Customer Churn Analysis Dashboard

## Overview

This project analyzes customer churn, retention, and revenue trends using a simulated **MTN Nigeria Customer Churn & Usage (Q1 2025)** dataset. The objective was to uncover the key factors influencing customer attrition and build an interactive Power BI dashboard that supports data-driven decision-making.

The workflow involved cleaning and preparing the data in **Python**, creating calculated fields and business metrics, and developing an executive dashboard in **Power BI** with custom DAX measures and interactive visualizations.

## Tools & Technologies

* Python (Pandas, NumPy)
* Google Colab
* Power BI
* DAX
* Power Query

## Dashboard Highlights

* Executive KPI cards with month-over-month comparisons
* Customer churn and retention analysis
* Revenue and customer growth trends
* Customer segmentation by age group and gender
* Churn analysis by state, device, and subscription plan
* Interactive slicers and reset filter functionality

## Key Insights

* Generated **₦199M** in total revenue from **496 customers**.
* Identified a **29.4% churn rate** and a **70.6% retention rate**.
* Highlighted customer segments, devices, and locations associated with higher churn.
* Provided insights to support targeted retention strategies and improve customer engagement.
